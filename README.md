# 🌐 Quipedia – Connecting Speakers and Colleges

**Quipedia** is a web platform designed to connect **institutions**, **students**, and **expert speakers** across various domains. Whether you're hosting or speaking, Quipedia helps simplify event management, increase visibility, and deliver real impact.

---

## 🚀 Live Preview

👉 [https://quipedia-client.onrender.com]([https://quipedia-client.onrender.com](https://quipedia-web-client.vercel.app/)) *(update this with your Render link if needed)*

---

## 📚 Project Summary

Quipedia empowers:

- **Colleges** to find verified speakers and student influencers.
- **Speakers** to get discovered and booked for talks with compensation.
- **Students** to host or attend certified knowledge-sharing sessions.

## 🧑‍💻 Getting Started (Local Dev)

### ✅ Prerequisites

- Node.js (v16+)
- npm or yarn

### ⚙️ Setup Instructions

```bash
# Clone the repo
git clone https://github.com/quipedia-work/quipedia_web_client.git
cd quipedia_web_client

# Install dependencies
npm install

# Start the development server
npm run dev

🔥 Features
🎙️ Speaker Discovery – Verified profiles with ratings, expertise, and availability

🏫 College Booking System – Post event requests and view proposals

🗓️ Smart Scheduling – Real-time calendar booking and reminders (coming soon)

📜 Certificates & Portfolio – Track sessions, feedback, and verified impact

👥 Role-based Auth – Separate views for students, colleges, and speakers (WIP)

🧪 Development Commands
Command	Purpose
npm run dev	Start development server
npm run build	Create production-ready build
npm run lint	Lint JavaScript with ESLint
npm run preview	Preview the production build
